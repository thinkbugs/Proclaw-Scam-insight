## 信任构建模型

### 时间序列模型

**信任值函数**：
```
T(t) = α × Σ(交互质量_i × 时间衰减_i) + β × 背书值 + γ × 展示价值
```

**参数说明**：
- T(t)：时刻t的信任值
- α：交互权重
- β：背书权重
- γ：展示价值权重
- 交互质量：每次互动的正面程度
- 时间衰减：越近的交互权重越高
- 背书值：他人背书的可信度
- 展示价值：展示的财富/能力

**骗局的信任构建策略**：

1. **高频低质量交互**（初期）：
   - 频率：每天多次
   - 质量：日常关心、问候
   - 目的：建立熟悉感

2. **中频中质量展示**（中期）：
   - 频率：每周几次
   - 质量：展示财富、能力
   - 目的：建立仰慕

3. **低频高质量转化**（后期）：
   - 频率：关键时刻
   - 质量：推荐"机会"
   - 目的：转化收割

## 群体行为传播算法

### 传播模型

**SIR模型变种**：
```
S(t) → I(t) → R(t)
S: 易感者（未接触骗局）
I: 感染者（已接触骗局）
R: 恢复者（已识别骗局或已被收割）
```

**传播方程**：
```
dS/dt = -β × S × I
dI/dt = β × S × I - γ × I
dR/dt = γ × I
```

**参数说明**：
- β：传播速率（骗局的吸引力）
- γ：恢复速率（识别骗局的速度）

**骗局传播特点**：
- β很高（骗子主动传播）
- γ很低（受害者难以识别）

### 网络效应算法

**梅特卡夫定律**：
```
网络价值 ∝ n²
n = 用户数量
```

**骗局应用**：
- 早期：快速拉人头
- 中期：制造"很多人在投"假象
- 后期：利用群体压力

**传播策略**：
1. **种子用户**：先发展10个核心
2. **快速复制**：每个核心再拉10个
3. **形成网络**：100人形成小网络
4. **网络效应**：100人可以影响1000人

---

### 羊群效应模型

**跟随概率**：
```
P(跟随) = α × P(信息性) + β × P(规范性) + γ × P(责任分散)
```

**参数说明**：
- 信息性：他人可能有更多信息
- 规范性：不想显得与众不同
- 责任分散：大家一起，责任共担

**骗局应用**：
- 制造"大家都在投"假象
- 展示虚假收益截图
- 利用社交压力

## 信息操纵技术

### 信息过滤算法

**信息茧房构建**：
```
只展示：成功案例、高收益、正面评价
隐藏：风险提示、负面信息、失败案例
```

**操控技巧**：
1. **选择性展示**：只展示好的一面
2. **信息延迟**：延迟负面信息
3. **信息混淆**：用复杂术语掩盖真相

---

### 叙事控制算法

**叙事公式**：
```
叙事 = 问题 + 解决方案 + 社会认同 + 紧迫感
```

**骗局叙事示例**：
- 问题："银行利息太低，通胀严重"
- 解决方案："我们的平台年化20%"
- 社会认同："已经有10万人投资"
- 紧迫感："名额有限，只剩3天"

---

### 信息不对称制造

**制造方法**：
1. **复杂化**：用复杂术语掩盖简单逻辑
2. **模糊化**：用模糊语言掩盖具体细节
3. **伪深度**：用深层概念包装浅层内容

## 心理操控框架

### 六大心理弱点

**弱点1：损失厌恶**
- 损失>收益（1:2.25）
- 利用："不投就亏了"

**弱点2：锚定效应**
- 第一印象影响后续判断
- 利用：先展示高收益，再降低

**弱点3：稀缺性偏误**
- 稀缺=有价值
- 利用："名额有限"

**弱点4：社会认同**
- 他人做的=正确的
- 利用："大家都在投"

**弱点5：权威偏误**
- 权威说的=正确的
- 利用：假冒专家

**弱点6：确认偏误**
- 只看支持自己观点的信息
- 利用：只展示正面信息

---

### 情绪操控算法

**贪婪激活**：
```
展示财富 → 激活贪婪 → 降低风险感知 → 增加投入
```

**恐惧激活**：
```
制造焦虑 → 激活恐惧 → 寻找安全感 → 接受叙事
```

**FOMO激活**：
```
制造稀缺 → 激活FOMO → 加速决策 → 交出资金
```

---

### 认知操控算法

**降低认知成本**：
```
简单化 → "你不用懂，跟着投就行"
权威化 → "专家已经验证过了"
案例化 → "看别人都赚了"
```

**制造认知错觉**：
```
确定性错觉 → "稳赚不赔"
控制错觉 → "我能控制风险"
知识错觉 → "我已经懂了"
```

---

### 时间操控算法

**时间压缩**：
```
制造紧迫感 → "只剩3天"
限制决策时间 → "现在不投就没了"
```

**时间延长**：
```
延长信任建立时间 → 长期聊天
延长回报等待时间 → "长期持有收益更高"
```

## 识别社交工程攻击

### 信任操控识别

**识别信号**：
- 突然示好
- 过度关心
- 快速建立亲密关系
- 展示虚假财富
- 引用虚假权威

**防御策略**：
- 验证身份
- 独立调查
- 保持警惕
- 不轻信"内部渠道"

---

### 信息操控识别

**识别信号**：
- 只展示正面信息
- 信息不透明
- 用复杂术语掩盖真相
- 拒绝回答具体问题

**防御策略**：
- 要求透明信息
- 要求独立验证
- 询问风险
- 不要被叙事迷惑

---

### 心理操控识别

**识别信号**：
- 制造紧迫感
- 制造稀缺性
- 利用群体压力
- 激活贪婪/恐惧

**防御策略**：
- 保持冷静
- 独立思考
- 不随大流
- 控制情绪
