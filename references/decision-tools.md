## 决策树

### 核心概念
决策树将复杂决策分解为简单的"是/否"问题，形成树状结构。

### 投资决策树

#### 第一层：能否回答"收益从哪来"？
- **否** → 拒绝
- **能** → 进入第二层

#### 第二层：能否回答"风险在哪"？
- **否** → 拒绝
- **能** → 进入第三层

#### 第三层：能否回答"谁承担风险"？
- **不知道** → 拒绝（你是风险承担者）
- **别人** → 进入第四层

#### 第四层：最坏情况能否接受？
- **不能** → 拒绝
- **能** → 进入第五层

#### 第五层：期望值是否为正？
- **否** → 拒绝
- **是** → 进入第六层

#### 第六层：仓位是否合理？
- **否** → 调整仓位
- **是** → 执行

### 骗局识别决策树

#### 第一层：是否有"确定性叙事"？
- **有**（"稳定收益""保本保息"） → 高风险，拒绝
- **无** → 进入第二层

#### 第二层：是否降低认知门槛？
- **是**（"不用懂""跟着就行"） → 高风险，拒绝
- **否** → 进入第三层

#### 第三层：是否建立信任闭环？
- **是**（小收益→加大投入） → 进入第四层
- **否** → 可能正常

#### 第四层：收益是否真实？
- **无法验证** → 拒绝
- **可验证** → 进入第五层

#### 第五层：资金是否可随时退出？
- **否**（锁定期、限制） → 拒绝
- **是** → 可能正常

### 退出决策树

#### 第一层：决策是否正确？
- **正确** → 继续持有
- **错误** → 进入第二层

#### 第二层：亏损是否可接受？
- **不可接受** → 立即止损
- **可接受** → 进入第三层

#### 第三层：是否有改进空间？
- **有** → 调整策略
- **无** → 进入第四层

#### 第四层：是否情绪驱动？
- **是** → 冷静后重新评估
- **否** → 接受结果

## 判断矩阵

### 骗局风险判断矩阵

| 判断维度 | 高风险特征 | 中风险特征 | 低风险特征 | 评分 |
|---------|-----------|-----------|-----------|------|
| 收益来源 | 说不清 | 依赖特定条件 | 明确逻辑 | 0-2 |
| 风险识别 | 承诺"无风险" | 风险模糊 | 风险明确 | 0-2 |
| 认知门槛 | "不用懂" | 需要专业分析 | 可自行理解 | 0-2 |
| 信任建立 | 小收益诱饵 | 无特殊机制 | 正常建立 | 0-2 |
| 流动性 | 锁定期 | 限制退出 | 随时退出 | 0-2 |
| 信息透明 | 不透明 | 部分透明 | 高度透明 | 0-2 |

**评分规则**：
- 高风险：0分
- 中风险：1分
- 低风险：2分

**总分**：0-12分

**风险等级**：
- 0-4分：极高风险 → 拒绝
- 5-8分：高风险 → 警惕
- 9-10分：中风险 → 审慎评估
- 11-12分：低风险 → 可考虑

## 风险评分系统

### 骗局风险评分（CRS）

**CRS = 0.3 × 收益来源 + 0.25 × 风险识别 + 0.2 × 认知门槛 + 0.15 × 流动性 + 0.1 × 信息透明**

**评分标准**（每项0-2分）：
- 0分：高风险特征
- 1分：中风险特征
- 2分：低风险特征

**风险等级**：
- CRS < 1.2：极高风险
- CRS 1.2-1.6：高风险
- CRS 1.6-2.0：中风险
- CRS > 2.0：低风险

### 流动性风险评分（LRS）

**LRS = 0.4 × 提现难度 + 0.3 × 锁定期 + 0.2 × 订单深度 + 0.1 × 滑点**

**评分标准**（每项0-3分）：
- 0分：极差
- 1分：差
- 2分：一般
- 3分：好

**风险等级**：
- LRS < 1.2：极高风险
- LRS 1.2-2.1：高风险
- LRS 2.1-2.7：中风险
- LRS > 2.7：低风险

### 综合风险评分

**综合风险 = 0.6 × CRS + 0.4 × LRS**

**风险等级**：
- 综合风险 < 1.4：拒绝
- 综合风险 1.4-1.8：谨慎
- 综合风险 > 1.8：可考虑

## 骗局识别检查清单

### 必查项（任一不符合 → 拒绝）

#### 收益相关
- [ ] 能否明确说出收益从哪里来？
- [ ] 收益是否符合市场平均水平？（不高于市场2倍）
- [ ] 收益是否可持续？（不是靠新资金兑付）

#### 风险相关
- [ ] 能否明确说出风险在哪里？
- [ ] 风险是否可控？（有明确的风控措施）
- [ ] 是否承诺"无风险"或"保本"？

#### 认知相关
- [ ] 是否能完全理解投资逻辑？
- [ ] 是否被告知"不用懂"？
- [ ] 是否需要依赖"专家"？

#### 流动性相关
- [ ] 资金是否可随时退出？
- [ ] 提现是否有限制？
- [ ] 是否有锁定期？

#### 信息相关
- [ ] 信息是否透明？
- [ ] 能否验证关键信息？
- [ ] 是否有独立的第三方验证？

### 可选项（多项不符合 → 警惕）

#### 信任相关
- [ ] 是否有权威背书？（机构、专家）
- [ ] 是否有成功案例？（可验证）
- [ ] 是否有历史业绩？（可追溯）

#### 情绪相关
- [ ] 是否被制造紧迫感？（"不投就晚了"）
- [ ] 是否被制造稀缺感？（"名额有限"）
- [ ] 是否被制造FOMO？（"错过就没了"）

#### 羊群相关
- [ ] 是否展示"大量用户"？（可验证）
- [ ] 是否展示"大V推荐"？（可验证）
- [ ] 是否制造"流行"假象？

### 检查清单使用规则

**规则1：必查项优先**
- 必查项任一不符合 → 立即拒绝
- 不看可选项，不浪费时间

**规则2：可选项辅助**
- 必查项全部通过 → 查看可选项
- 可选项多项不符合 → 警惕，可能隐藏风险

**规则3：独立验证**
- 不轻信对方提供的"验证"
- 使用独立的数据源
- 使用第三方验证

### 模板使用规则

**规则1：完整性**
- 所有必填项必须填写
- 任何一项无法填写 → 拒绝

**规则2：独立性**
- 独立填写，不受对方影响
- 使用独立数据源
- 不轻信对方提供的信息

**规则3：诚实性**
- 承认无知
- 不强求答案
- 不确定就写"不知道"

**规则4：纪律性**
- 严格执行结论
- 不违反规则
- 不情绪化决策

### 决策流程

#### Step 1：信息收集（30分钟）
- 阅读项目资料
- 收集独立信息
- 验证关键数据

#### Step 2：核心问题回答（30分钟）
- 回答五个核心问题
- 验证每个答案
- 记录不确定性

#### Step 3：量化评估（1小时）
- 计算期望值
- 计算凯利公式
- 计算夏普比率
- 计算最大回撤

#### Step 4：风险评分（30分钟）
- 评估骗局风险
- 评估流动性风险
- 计算综合风险

#### Step 5：决策结论（15分钟）
- 做出最终决策
- 确定仓位
- 记录决策理由

#### Step 6：延迟确认（24小时）
- 等待24小时
- 冷静后重新评估
- 确认决策不变后执行

## 决策工具的终极洞察

### 核心真相
> 好的决策工具不是"让你更聪明"，而是"让你更理性"

### 决策工具的价值
- **减少情绪干扰**：用工具替代直觉
- **标准化流程**：避免遗漏关键步骤
- **可追溯性**：记录决策过程
- **可重复性**：相同输入产生相同输出

### 顶级玩家的决策习惯
- 使用系统化工具
- 延迟决策
- 独立思考
- 持续学习
- 复盘总结

### 决策工具的局限
- 工具只是辅助，不能替代思考
- 工具需要正确使用
- 工具需要不断优化
- 工具不能消除不确定性

### 最后的建议
> 使用工具，但不依赖工具
> 遵循流程，但不盲从流程
> 相信系统，但不放弃思考
