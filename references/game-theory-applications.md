## 概览

骗局是博弈论的典型应用，理解博弈论可以帮助识别和防御骗局。

### 核心概念

1. **纳什均衡**：没有任何一方愿意单独改变策略的状态
2. **零和博弈**：一方收益=另一方损失
3. **非零和博弈**：双方可能都收益或都损失
4. **信息不对称**：一方拥有另一方没有的信息

## 纳什均衡分析

### 骗局中的纳什均衡

**场景**：投资者 vs 骗子

**策略矩阵**：

| | 投资者投 | 投资者不投 |
|---|---|---|
| **骗子真实运营** | 投资者：赚<br>骗子：赚 | 投资者：0<br>骗子：亏 |
| **骗子跑路** | 投资者：亏<br>骗子：大赚 | 投资者：0<br>骗子：亏 |

**纳什均衡**：
- 骗子的最优策略：跑路
- 投资者的最优策略：不投

**启示**：
- 不投资是投资者的最优策略
- 假设骗子会跑路是正确的

---

### 囚徒困境应用

**场景**：两个投资者，一个跑路

**策略矩阵**：

| | 投资者A跑路 | 投资者A不跑 |
|---|---|---|
| **投资者B跑路** | A：小亏<br>B：小亏 | A：大亏<br>B：小亏 |
| **投资者B不跑** | A：小亏<br>B：大亏 | A：大亏<br>B：大亏 |

**纳什均衡**：
- 两人都跑路

**启示**：
- 崩盘时，跑路是理性选择
- 先跑者优势

## 博弈树构建

### 骗局博弈树

```
骗子决策
├─ 选择策略
│  ├─ 真实运营
│  │  └─ 投资者决策
│  │     ├─ 投资者投 → 长期收益
│  │     └─ 投资者不投 → 骗子亏
│  └─ 跑路
│     └─ 投资者决策
│        ├─ 投资者投 → 骗子大赚，投资者亏
│        └─ 投资者不投 → 双方都亏
```

**逆向归纳**：
- 投资者：假设骗子会跑路 → 不投
- 骗子：知道投资者会不投 → 跑路

**启示**：
- 不要假设骗子会诚实
- 最坏情况分析

---

### 投资者决策树

```
遇到投资机会
├─ 检查收益率
│  ├─ >15% → 高风险 → 不投
│  └─ ≤15% → 继续检查
├─ 检查资金托管
│  ├─ 无托管 → 不投
│  └─ 有托管 → 继续检查
├─ 检查收益来源
│  ├─ 不明 → 不投
│  └─ 明确 → 继续检查
├─ 小额试水
│  ├─ <1%资金
│  └─ 观察1-3个月
└─ 决策
   ├─ 正常提现 → 谨慎考虑
   └─ 提现困难 → 立即撤出
```

## 多轮博弈策略

### 重复博弈

**无限重复博弈**：
- 如果博弈无限重复，合作可能成为均衡
- 骗子可能选择长期经营

**有限重复博弈**：
- 如果博弈有限，最后一轮会选择背叛
- 骗子会选择在最后一轮跑路

**启示**：
- 不要相信"长期经营"的承诺
- 假设博弈是有限的

---

### 触发策略

**触发策略**：
- 开始时合作
- 对方背叛后，永远背叛

**应用**：
- 平台正常兑付 → 继续
- 平台违约 → 立即撤出，永不回头

**启示**：
- 一次违约=永远拒绝
- 不给第二次机会

---

### 混合策略

**混合策略**：以一定概率选择不同策略

**应用**：
- 投资：小额试水（10%投，90%观望）
- 退出：观察信号（50%退出，50%继续）

**启示**：
- 不all in
- 保持灵活性

## 博弈论防御策略

### 最坏情况分析

**原则**：
- 假设骗子会跑路
- 假设信息是虚假的
- 假设最坏情况会发生

**应用**：
- 不依赖"承诺"
- 不相信"长期"
- 不假设"例外"

---

### 逆向选择分析

**逆向选择**：坏的产品驱逐好的产品

**应用**：
- 好项目不需要高收益宣传
- 高收益宣传的都是坏项目
- 市场充斥坏项目

**启示**：
- 高收益=骗局
- 真正的好项目不会被你遇到

---

### 道德风险防范

**道德风险**：一方有动机损害另一方

**应用**：
- 骗子有动机跑路
- 投资者无法控制骗子

**启示**：
- 把控制权掌握在自己手中
- 不信任任何"保证"
