## 核心理论体系 / Core Theoretical Framework

### 第一层：底层公理 / Layer 1: Axioms

**四大底层公理 / Four Fundamental Axioms**

1. **收益分配公理 / Revenue Allocation Axiom**
   - 收益不是创造的，是分配的
   - 要么来自别人亏损，要么来自未来（时间/风险透支）
   - Revenue is allocated, not created
   - Either from others' losses or from future (time/risk overdraft)

2. **风险转移公理 / Risk Transfer Axiom**
   - 风险不会消失，只会转移
   - 谁没看见风险，谁就是风险承接者
   - Risk never disappears, only transfers
   - Whoever doesn't see risk becomes the bearer

3. **认知优势公理 / Cognitive Advantage Axiom**
   - 市场不是公平系统，是认知与结构优势系统
   - Market is not fair, but a system of cognitive and structural advantage

4. **不确定性公理 / Uncertainty Axiom**
   - 不确定性是底层常量，不能被消除
   - Uncertainty is the underlying constant and cannot be eliminated

### 第二层：核心轴 / Layer 2: Core Axes

**三大核心轴 / Three Core Axes**

1. **不确定性 vs 确定性幻觉 / Uncertainty vs Certainty Illusion**
   - 顶层：利用不确定性
   - 底层：逃避不确定性

2. **认知深度 vs 叙事依赖 / Cognitive Depth vs Narrative Dependency**
   - 顶层：看结构
   - 底层：听故事

3. **控制权 vs 被控制 / Control vs Being Controlled**
   - 顶层：自己决策
   - 底层：交出决策权

### 第三层：定价权 / Layer 3: Pricing Power

**三大核心权力 / Three Core Powers**

1. **定价权 / Pricing Power**: 定义什么是价值
2. **流量控制权 / Flow Control Power**: 控制资金流向
3. **规则定义权 / Rule Definition Power**: 设计游戏机制

**终极公式 / Ultimate Formula**:
```
定价权 = 解释权 = 收益分配权
```

## 五步强制判断流程 / Five-Step Mandatory Decision Process

### Step 1: 识别需求缺口 / Identify Demand Gap

**核心问题 / Core Question**: 
- 为什么会出现这个"机会"？
- 这个"机会"在解决谁的焦虑？

**判断标准 / Criteria**:
- ❌ 制造焦虑（"错过就没了"）
- ❌ 夸大风险（"银行不安全"）
- ❌ 紧迫感（"限时优惠"）

### Step 2: 分析确定性叙事 / Analyze Certainty Narrative

**核心问题 / Core Question**:
- 它是如何提供"确定性"的？
- 这个"确定性"违背了什么公理？

**判断标准 / Criteria**:
- ❌ 稳定收益（"保本保息"）
- ❌ 低风险高回报（"年化30%保本"）
- ❌ 无风险套利（"稳赚不赔"）

### Step 3: 拆解风险结构 / Decompose Risk Structure

**必须回答的五个问题 / Five Must-Answer Questions**:

1. **收益从哪里来？/ Where does revenue come from?**
   - 如果不知道 → ❌ 立即拒绝

2. **风险在哪里？/ Where is the risk?**
   - 如果看不到风险 → ❌ 你就是风险承接者

3. **谁在承担风险？/ Who bears the risk?**
   - 如果不是你 → ❌ 风险会转移给你

4. **最坏情况是什么？/ What's the worst case?**
   - 如果无法承受 → ❌ 不要参与

5. **如果我错了，会不会死？/ Will I die if wrong?**
   - 如果会 → ❌ 立即退出

### Step 4: 识别收割节点 / Identify Harvesting Nodes

**检查是否进入收割路径 / Check if entering harvesting path**:

- ✓ 小额试水 → 给你收益 → 强化信任 → 加大投入
- ✓ 信任建立 → 情绪放大 → 从众出现
- ✓ 资金聚集 → 临界点 → 风险转移

**危险信号 / Danger Signals**:
- 鼓励加仓 / Encourage increasing position
- 推荐亲友 / Recommend to friends
- 承诺保本 / Promise capital protection
- 提现困难 / Withdrawal difficulties

### Step 5: 执行反收割决策 / Execute Anti-Harvest Decision

**决策矩阵 / Decision Matrix**:

| CRS评分 | LRS评分 | 决策 / Decision |
|--------|--------|----------------|
| 0-3分 / 0-3 pts | 0-4分 / 0-4 pts | ✓ 可以考虑 / Consider |

### 案例一：e租宝 - P2P爆雷 / Case 1: eZubao - P2P Collapse

**背景 / Background**:
- 时间：2014-2015年
- 涉案金额：500亿人民币
- 受害人数：90万+

**骗局结构 / Scam Structure**:
1. 制造需求：中小企业融资难
2. 提供方案：年化9%-14.6%的"安全理财"
3. 降低门槛：1元起投，随时提现
4. 建立信任：央视广告、明星代言
5. 资金聚集：90万投资者，500亿资金
6. 风险转移：虚假标的，资金挪用
7. 崩盘：2015年12月被查封

**识别点 / Identification Points**:
- ❌ 收益率远超市场合理水平
- ❌ 大规模央视广告（成本回收压力大）
- ❌ 虚假标的（借款企业多为空壳）
- ❌ 资金挪用（未进入第三方托管）

**教训 / Lessons**:
- 高收益必然伴随高风险
- 广告≠靠谱
- 必须审查底层资产真实性

### 案例二：麦道夫 - 史上最大庞氏 / Case 2: Madoff - Largest Ponzi

**背景 / Background**:
- 时间：1992-2008年
- 涉案金额：650亿美元
- 受害人数：4,800+

**骗局结构 / Scam Structure**:
1. 制造需求：稳定收益的"华尔街传奇"
2. 提供方案：年化10%-12%的"稳定回报"
3. 降低门槛：只接受"熟人"推荐
4. 建立信任：前SEC主席背书，名人客户
5. 资金聚集：4,800投资者，650亿美元
6. 风险转移：用新投资者资金支付老投资者
7. 崩盘：2008年金融危机，资金链断裂

**识别点 / Identification Points**:
- ❌ 长期稳定收益（市场不可能）
- ❌ 不公布投资策略
- ❌ 不接受外部审计
- ❌ 只接受"熟人"（控制流动性需求）

**教训 / Lessons**:
- 长期稳定收益是最大的红旗
